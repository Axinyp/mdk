# Stage 5 UI 设计规范

## 设计系统基准

- **字体**: Plus Jakarta Sans 300/400/500/600/700
- **主色**: `#64748B`（工业灰）
- **强调色/CTA**: `#F97316`（安全橙）
- **背景**: `#F8FAFC`（极浅灰）
- **正文**: `#334155`（深蓝灰）
- **辅助色**: `#94A3B8`（中灰）
- **边框**: `border-slate-200`
- **风格**: Micro-interactions（小悬浮动画、状态反馈、50-100ms hover）
- **设计文件**: `design-system/mkcontrol-mdk/MASTER.md`
- **参考栈**: React + TypeScript + Tailwind CSS

### 消息气泡设计

```

┌─────────────────────────────────────────────────┐
```

## 全局交互规范（所有新页面适用）

| 规则 | 实现 |
|------|------|
| 流式文本 | 逐字渲染，光标闪烁；`prefers-reduced-motion` 直出 |
| 加载按钮 | 发送期间 `disabled + opacity-50`，内嵌 spinner（w-4 h-4） |
| Toast 通知 | 右下角，`bg-slate-800 text-white`，3s 后自动消失 |
| 错误状态 | 就近提示，`bg-red-50 border-red-200 text-red-700` |
| 空状态 | 居中图标 + 说明文字 + CTA，不显示空表格 |
| 骨架屏 | `animate-pulse bg-slate-200 rounded`，替代 spinner 用于内容加载 |
| 抽屉动画 | `translate-x-full → translate-x-0`，`duration-300 ease-out` |
| Modal 背景 | `bg-black/40 backdrop-blur-sm` |
