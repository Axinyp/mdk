**DEFINE_VARIABLE: 变量定义块**
**变量定义必须放在此块中。**
**变量定义格式：**
```
变量类型 变量名 = 初值;
```
> ⚠️ **约束：所有变量必须赋初值，不能只声明不赋值。**

**例子:**
```
DEFINE_VARIABLE
    int x = 222;              // 定义整形变量并初始化
    double f = 2.5;           // 定义双精度浮点变量并初始化
    string s = "hello";       // 定义字符串变量并初始化
    char c = 'A';             // 定义字符变量并初始化
    boolean b = false;        // 定义布尔变量并初始化
    byte w = 0;               // 定义字节变量并初始化
```
