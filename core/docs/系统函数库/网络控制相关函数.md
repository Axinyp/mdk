# SEND_TCP
```
void SEND_TCP(String ip,int port,String str)
```
 功能：以TCP连接方式向主机发送数据
 **Parameters:**
 `ip` :主机IP地址
 `port` :主机端口号
 `str` :要发送的数据串
示例：
```
SEND_TCP("192.168.1.20", 2000, "0123456789"); //向IP地址为192.168.1.20，端口号为2000的主机发送数据
```

# WAKEUP_ONLAN
```
void WAKEUP_ONLAN(String MAC)
```
功能：PGMIII通过局域网唤醒电脑。可用于唤醒处于关闭或睡眠状态的电脑。
**Parameters:**
`MAC` :主机MAC地址
示例：
```
WAKEUP_ONLAN("4437e65b1735"); //唤醒MAC地址为44-37-e6-5b-17-35的主机       
```
