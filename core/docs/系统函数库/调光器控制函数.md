# SEND_LITE
```text
void SEND_LITE(String dev,int channel,int val)
```
功能：控制灯光
**Parameters:**
`dev` \- :灯光设备
`channel` \- :设备通道号
`val` \- ：模拟量 （注：根据具体外接设备取值。）
示例：
```
lite_n = N:8:LITE;; //定义CRNET号为8上的的灯光设备
SEND_ LITE (lite_n,1,65535); //向lite_n的第一路发送模拟量65535
```

# Z_SEND_LITE
```
void Z_SEND_LITE(String dev, int channel, int val)
```
 功能：向调光器发送控制指令
 **Parameters:**
`dev` ：设备 
`channel` ：控制第几路灯光
`val` ：模拟值 0-65535
示例：
```
LITE_Z = Z:10:ZLITE:"0102030405"; //定义ZigBee卡ID为10的调光器
Z_SEND_LITE(LITE_Z, 2,1000); //设定调光器第二路亮度为1000（模拟值）
```
* * *
# Z_SEND_WMLITE
```
void Z_SEND_WMLITE(String dev, int channel, int val)
```
* 功能：向触摸按键调光器发送控制指令
* param dev ：设备
* param channel ：控制第几路灯光 范围：1-2
* param val ：模拟值 0-65535
示例：
```
WMLITE_Z = Z:10:WMZLITE:"0102030405"; //定义ZigBee卡ID为10的按键调光器
Z_SEND_WMLITE(WMLITE_Z, 2,1000); //设定按键调光器第二路亮度为1000（模拟值）
```
注：WMLITE设备读取按键状态还是用Z_READ_WM函数。

# Z_QUERY_LITE
```
void Z_QUERY_LITE(String dev, int channel)
```
* 功能：向电源控制器发送查询指令
* param dev ：设备
* param channel ：获取第几路灯光值 
示例： 
```
LITE_Z = Z:10:ZLITE:"0102030405"; //定义ZigBee卡ID为10的调光器
Z_QUERY_POWER(LITE_Z,4); //向调光器发送查询信息，查询第四路亮度
```
