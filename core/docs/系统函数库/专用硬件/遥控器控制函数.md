# Z_SEND_RC
```
void Z_SEND_RC(String dev, int onoff, int channel)
```
 功能：向遥控器发送控制指令
**Parameters:**
`dev` ：设备
`onoff` ： 打开或关闭LED,1 打开, 0 关闭
`channel` ：打开或关闭哪一路LED 范围：1-8
示例：
```
RC_Z = Z:10:ZRC:"0102030405"; //定义ZigBee卡ID为10的遥控器
Z_SEND_RC(RC_Z , 1, 4); //打开第四路LED
```

# Z_QUERY_RC
```
void **Z_QUERY_RC**(String dev)
```
功能：向遥控器发送查询指令
 **Parameters:**
 `dev` ：设备
示例：
```
RC_Z = Z:10:ZRC:"0102030405"; //定义ZigBee卡ID为10的遥控器
Z_QUERY_RC(RC_Z ); //向遥控器发送查询信息
```

# Z_READ_RC
```
int Z_READ_RC( byte[] data)
```
功能：读取遥控器按键状态
返回值：1 or 0。1 按下, 0 弹起。若没有匹配的SN号则返回-1
示例：
```
int i; //建议将i初始化为0,1以外的数
i = Z_READ_RC( DATA.Data); //读取遥控器返回的按键状态
```
