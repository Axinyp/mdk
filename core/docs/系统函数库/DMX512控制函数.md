#   SEND_DMX512
```text
void SEND_DMX512(String dev, int channel, int val)
```
功能：控制DMX512
**Parameters:**
`dev` \- 输入设备
`channel` \- :设备通道号
`val` \- :模拟量（注：根据具体外接设备取值。）
示例：
```
lite_n = N:8:LITE;
SEND_ LITE (lite_n,1,65535);
```
