# SAVE_PARAM
```
void SAVE_PARAM(String Name, int Param)
```
功能：存储变量，每Save一次都会覆盖上一次的值。
**Parameters:**
 `Name` :变量名。 
 `Param`:变量本身（该参数支持的类型：int、double、string、int[]、double[]、string[]）。   
示例： 
```text
int Num = 100;	
double PI = 3.14;	
string str = " ";  
 		
int nArray[10];	
double dArray[10];	
string strArray[10]; 	

SAVE_PARAM("Num", Num);   //存储Num变量	
SAVE_PARAM("PI", PI);	
SAVE_PARAM("str", str); 	
SAVE_PARAM("nArray", nArray);   //存储nArray数组	
SAVE_PARAM("dArray", dArray);	
SAVE_PARAM("strArray", strArray); 
```
# LOAD_PARAM
```
void LOAD_PARAM(String Name, int Param) 
```
* 功能：读取变量。    
**Parameters:**
`Name` :变量名。 
`Param`:变量本身（该参数支持的类型：int、double、string、int[]、double[]、string[]）。 
 返回值类型：可以是 int、double、string、int[]、double[]、string[]。  
 示例：
```
int Num = 100; 
double PI = 3.14;
string str = " "; 

int nArray[10];
double dArray[10]; 
string strArray[10];  
Num = LOAD_PARAM("Num", Num);   //读取Num变量的值并赋值
PI = LOAD_PARAM("PI", PI); 
str = LOAD_PARAM("str", str);

nArray = LOAD_PARAM("nArray", nArray);   //读取nArray数组的值并赋值 
dArray = LOAD_PARAM("dArray", dArray); 
strArray = LOAD_PARAM("strArray", strArray); 

```

# DEL_ALL_PARAM  
 void DEL_ALL_PARAM() 
 功能：删除所有存储的变量信息。       
 示例：  
```
DEL_ALL_PARAM(); 
```
