# SOUND_PLAY
```
void**SOUND_PLAY**(String dev, int id)
```
* 功能：播放指定曲目
**Parameters:**
`dev`：设备名
`id`：要播放曲目的编号，1-65535。
示例：
```
SOUND_Z= Z:10:SOUND:"0102030405"; //定义ZigBee卡ID为10的美声器
SOUND_PLAY(SOUND_Z, 114) ;//播放曲目为第114首
```

# SOUND_SETLOW
```
void SOUND_SETLOW(String dev,int lowMax, int Gain)
```
 功能：美声器低音设置
`dev`：设备
`lowMax`：低音上限 2-15（20HZ-150HZ）
`Gain`：低音的增益范围 0-15（0DB-15DB）
示例：
```
SOUND_Z = Z:10:SOUND:"0102030405"; //定义ZigBee卡ID为10的美声器
SOUND_SETLOW(SOUND_Z, 2, 2);
```
* * *

# SOUND_SETHIGHT
```
void SOUND_SETHIGHT(String dev,int highMin, int Gain)
```
 功能：美声器低音设置
 **Parameters:**
`dev`：设备名
`highMin`：高音下限 1-15（1K-15K）
`Gain`：高音的增益范围 0-15 （0=关闭增益,0to10.5DB,-1.5DBto-12DB） 步进1.5DB
示例：
```
SOUND_Z = Z:10:SOUND:"0102030405"; //定义ZigBee卡ID为10的美声器
SOUND_SETHIGHT(SOUND_Z, 15, 15);
```
