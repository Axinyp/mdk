# ## 
UP_WM
```text
  void UP_WM(String dev,int channel)
```

功能：给墙上面板发送弹起信息 ，用于触摸屏和墙上面板控制同一设备时的消息交互。
**Parameters:**
`dev `\- : 面板设备
`channel` \- :设备通道号
示例：
```
wm_N = N:14:WM; // 定义CRNET设备号为14的墙上面板设备
UP_WM(wm_N,1);//要求wm_N设备的第一路是弹起状态
```

`channel` \- :设备通道号

DOWN_WM(wm_N,1);//要求wm_N设备的第一路是按下状态

# Z_QUERY_WM
```
void Z_QUERY_WM(String dev) 
```
功能：向墙上面板发送查询指令
**Parameters:**
`dev` ：设备
示例：
```
WM_Z = Z:10:ZWM:"0102030405";//定义ZigBee卡ID为10的墙上面板 
Z_QUERY_WM(WM_Z ); //向墙上面板发送查询信息
```

# Z_READ_WM
```
int ZIGBEE_READ_WM( byte []data) 
```
功能：读取墙上面板按键状态
返回值：1 or 0。1 按下, 0 弹起。若没有匹配的SN号则返回-1
示例： 
```
int i;i = Z_READ_WM( DATA.Data); //读取墙上面板返回的按键状态
```
