#  DEV_QUERY
```text
  void DEV_QUERY(String dev, int channel)
```

功能：设备查询 ，功能：设备登记
	**Parameters:**
	`dev` \- 输入设备
	`channel` \- :设备通道号
示例：

```text
wm_N = N:14:WM; // 定义CRNET设备号为14的墙上面板设备
DEV_ QUERY (wm_N,1);//查询墙上面板wm_N的第一路
```
