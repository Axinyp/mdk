**Z_QUERY_TC**
void **Z_QUERY_TC**(String dev, int channel)* 功能：向传感器发送查询指令* param dev ：设备
* param channel ：传感器通道号
示例：TC_Z = Z:10:ZTC:"0102030405"; //定义ZigBee卡ID为10的传感器Z_QUERY_TC(TC_Z, 1 ); //查询当前温度信息
Z_QUERY_TC(TC_Z, 2 ); //查询当前湿度信息
Z_QUERY_TC(TC_Z, 3 ); //查询当前光照强度
Z_QUERY_TC(TC_Z, 4 ); //查询有无烟雾

**Z_READ_TEMPER**
double**Z_READ_TEMPER**( byte []data)* 功能：读取传感器返回的温度，单位--摄氏度，精度--小数点后一位。如：26.1 ℃。* 返回值：双精度型，实际温度。错误返回0.01（鉴于温度返回值比较特殊，所以以0.01做为错误号）
示例：double d; d = Z_READ_TEMPER( DATA.Data); //读取传感器返回的温度
* * *
**Z_READ_HUMID**
double **Z_READ_HUMID**( byte []data)* 功能：读取传感器返回的湿度，单位--百分比，精度--小数点后一位。如：41.2%。* 返回值：双精度型，湿度。错误返回-1
示例：double d; d = Z_READ_HUMID( DATA.Data); //读取传感器返回的湿度
* * *
**Z_READ_FOG**
int **Z_READ_FOG**( byte []data)* 功能：读取传感器返回的烟雾测试值。* 返回值：整型，烟雾测试值。返回0：无烟雾。返回1：有烟雾。错误返回-1
示例：int d; d = Z_READ_FOG( DATA.Data); //读取传感器返回的烟雾测试值
* * *
**Z_READ_LIGHT**
int**Z_READ_LIGHT**( byte []data)* 功能：读取传感器返回的光照强度，单位--勒克斯。如：800lux。* 返回值：整型，光照强度。错误返回-1
示例：int d; d = Z_READ_LIGHT( DATA.Data); //读取传感器返回的光照强度
