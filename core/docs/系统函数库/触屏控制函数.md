# 触屏控制函数 (help)
以下函数仅适用于支持双向的触屏设备。

---

# SET_BUTTON
```
void **SET_BUTTON**(String dev, int channel, int state)
```
功能：设置按钮状态
param dev：设备名
param channel：通道号
param state：发送按钮状态，1 or 0。1表示按下，0表示弹起。
示例：
```
tp = T:10:TP;
SET_BUTTON(tp, 1, 1); //将通道号为一的按钮状态设置为按下。
```
* * *
# SET_LEVEL
```
void **SET_LEVEL**(String dev, int channel, int val)
```
功能：设置拉条值
param dev：设备名
param channel：通道号
param val：拉条值. 0-65535
示例：
```
tp = T:10:TP;
SET_LEVEL(tp, 1, 255); //将通道号为一的拉条的值设为255
```
* * *
**SEND_TEXT**
void **SEND_TEXT**(String dev, int channel, String text)
功能：发送文本
param dev：设备名
param channel：通道号
param text：文本内容
示例：
```
tp = T:10:TP;
SEND_TEXT(tp, 1, "Hello");
```
* * *
# SEND_PAGING
```
void SEND_PAGING(String dev, int channel, String text)
```
功能:触摸屏界面翻页
**Parameters:**
`dev` \-：设备名
`channel` \-：1
`text` \-：跳页字符串
示例：
```
tp = T:10:TP;
SEND_PAGING(tp, 1, “矩阵控制”); //跳转到矩阵控制页面
```
