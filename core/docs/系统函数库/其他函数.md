# TRACE
```
void **TRACE**(string msg)
```
功能：打印消息msg
参数：msg - : 字符串表示消息的内容
示例：
```
TRACE("This is my program!")；
打印字符串  "This is my program!"
```
***
# RANDOM_NUMBER
```text
int RANDOM_NUMBER(int n)
```
* 功能：产生一个随机数       
* param n :随机数范围    
* 返回值：
* 0-n范围内的随机数
示例：
```
int i;
i = RANDOM_NUMBER(100); //产生0-100的随机数。
```

# GET_VER_INFO   	
```
string GET_VER_INFO ()        
```
功能：获取AllInOne版本信息        
返回值：字符串形式的版本信息。	
示例：	
```
string str;	
str = GET_VER_INFO(); //可以使用SEND_UDP函数将结果发送出来
```
