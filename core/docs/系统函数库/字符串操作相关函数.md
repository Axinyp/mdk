# BYTES_TO_STRING
```text
  String BYTES_TO_STRING(byte[] bb)
```
**功能 ：字节转换为字符串**
**Parameters:**
`bb`:字节数组对象。
**Returns:**
系统默认编码字符串。
示例：
```
DEFINE_VARIABLE
	string tt;
	string xxx;
	byte bb[0];
DEFINE_EVENT
	BUTTON_EVENT(TP1,2)
	{
		PUSH()
		{
			tt = "xxxxx123sdf456";
			bb = STRING_TO_BYTES(tt); 
			xxx = BYTES_TO_STRING(bb); 
			TRACE(xxx );
		}
	}
```
(注：这里的变量建议在**DEFINE_VARIABLE** 标签下先定义)

# STRING_EQ
```text
boolean STRING_EQ(String src,String des)
```
功能 ：两个字符串比较，严格按大小写比较
**Parameters:**
`src` **-** 比较字符1
`des` **-** 比较字符2
**Returns:**
src 等于 des 返回 `true`，其他返回`false`
示例：
```
DEFINE_VARIABLE
	string t1;
	string t2;
DEFINE_EVENT
	BUTTON_EVENT(TP1,2)
	{
		PUSH()
		{
			t1 = "123";
			t2 = "122";
			if( STRING_EQ(t1,t2) )
			{
				TRACE("t1 == t2" );
			}
			else
			{
				TRACE("t1 != t2" );
			}
		}
	}
// 按下触屏TP1的joinmuber =2 的按钮时输出如下结果
// t1 != t2
```

# BYTES_ADD
```text
  byte[] BYTES_ADD(byte[] first,byte[] second)
```
功能：将参数2加到参数1的尾部，组成一个新的字节数字返回。
**Parameters:**
`first` \- ：参数1,字节数组
`second` \- :参数2,字节数组
**Returns:**
：返回新的字节数组
示例：
```
DEFINE_VARIABLE 
	byte c1Bytes[0];
	int c1Blen;
DEFINE_START
	RESET_BYTE( c1Bytes );
DEFINE_EVENT
	DATA_EVENT(com_M,1)
	{
		ONDATA()
		{
			//将收到的串口1数据不断的加到c1Bytes尾。(这里的DATA.Data为内部DATA_EVENT的内部对象,为串口的数据)
			c1Bytes= BYTE_ADD(c1Bytes,DATA.Data); 
			//注：返回的才是两个字节数组相加的结果
			c1Blen = GET_BYTES_LENGTH(c1Bytes); // 获取c1Bytes的长度
			if(c1Blen > 30 )
			{
				// 处理主机板上串口1的数据
				TRACE(BYTES_TO_HEX(c1Bytes));
				//将c1Bytes清空，接受下一次到达数据,否则c1Bytes一直将串口数据加到尾
				RESET_BYTE( c1Bytes );
			}
		}
	}
```

# GET_SUB_STRING
```
String GET_SUB_STRING(String src, int beginIndex)
```
返回一个新的字符串，它是此字符串的一个子字符串。该子字符串从指定索引处的字符开始，直到此字符串末尾。 
示例： 
```
GET_SUB_STRING("unhappy", 2); returns "happy"
GET_SUB_STRING("Harbison", 3); returns "bison"
```
**参数**： beginIndex - 起始索引（包括）。 
**返回：** 指定的子字符串。 
```
String GET_SUB_STRING(String src, int beginIndex, int endIndex)
```
返回一个新字符串，它是此字符串的一个子字符串。该子字符串从指定的 `beginIndex` 处开始，直到索引 `endIndex - 1` 处的字符。因此，该子字符串的长度为 `endIndex-beginIndex`。 
示例： 
```
 GET_SUB_STRING("hamburger", 4, 8); returns "urge"
 GET_SUB_STRING("smiles", 1, 5); returns "mile"
```
**参数** ： 
`beginIndex` \- 起始索引（包括）。 
`endIndex` \- 结束索引（不包括）。     
**返回** ： 指定的子字符串。

# BYTES_TO_INT
```
int BYTES_TO_INT (byte[] b)
```
功能：将byte数组的前4位当做一个int数值处理，如b小于4位，则按多少位转换，大端模式。
**Returns**：返回转换后的int型数值
```
DATA_EVENT(mcar,2)
{
  ONDATA()
  {
  double dmv = BYTES_TO_INT(DATA.Data)                            
  SEND_COM(COM,1,DOUBLE_TO_STRING(dmv));
  }
}
```
