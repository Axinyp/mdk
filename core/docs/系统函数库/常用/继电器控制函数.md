# 继电器控制函数

**ON_RELAY OFF_RELAY QUERY_RELAY **
# ## 
* * *
# ## ON_RELAY
```text
void ON_RELAY(String dev,int channel)
```

功能：打开继电器
**Parameters:**
`dev` \- :继电器设备
`channel` \- :设备通道号
示例：
```
RELAY_M = M:1000:RELAY; //定义主机板号为1000的继电器
ON_RELAY(RELAY_M,2); // 打开主机板号为1000上的继电器的第2路。
```
* * *
# ## OFF_RELAY
```text
void OFF_RELAY(String dev,int channel)
```

功能：关闭继电器
**Parameters:**
`dev` \- ：继电器设备
`channel` \- :设备通道号
示例：
```
RELAY_M = M:1000:RELAY; //定义主机板号为1000的继电器
OFF_RELAY(RELAY_M,2); // 关闭主机板号为1000上的继电器的第2路。
```

****QUERY_RELAY****
void **QUERY_RELAY**(String dev,int channel)
* 功能：向继电器发送查询指令
* param dev ：设备名 
* param channel：通道号
示例：
``` cht
RELAY_N = N:7:RELAY; //定义CRNET号为7的继电器
QUERY_RELAY(RELAY_N, 1) //查询CRNET号为7的继电器的第一路负载状况 

//发送指令后将，继电器回复数据将触发数据事件，在数据事件中读取即可。
DATA_EVENT(RELAY_N,1) 
{ 
 ONDATA() 
 { 
  if (DATA.Data[0] == 0) //返回值=0 无负载 
  { 
   ON_RELAY(RELAY_N, 1); 
  } 
  if (DATA.Data[0] == 1) //返回值=1 有负载 
  { 
   OFF_RELAY(RELAY_N, 1); 
  }
 } 
}
```
