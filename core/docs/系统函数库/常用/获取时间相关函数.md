# GET_YEAR
```text
  int GET_YEAR()
```
功能：获取当前系统时间的年
**Returns:**
	返回int型的年，如2010
```text
示例：
int  year = GET_YEAR();
```

#  GET_DAY
```text
  int GET_DAY()
```

功能：获取当前系统今天是星期几
Returns: 
：返回int型的星期几（ 0-6 ）。星期天 = 0、星期一 = 1 ........以此类推。
示例：
```text
int  today = GET_DAY();
```
