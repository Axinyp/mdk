### SET_COM
```text
void SET_COM(String dev,int channel, long sband,int databit, int jo,int stopbit,int dataStream, int comType)
```
功能：设置COM口
**Parameters:**
`dev` \- :设备名称
`channel` \- :设备通道号
`sband` \- ：波特率
`databit` \- ：数据位 1~8 
`jo` \- ：奇偶位 0：无，1：奇数，2：偶数，3：标记，4：空格
`stopbit`-：停止位10,15,20,对应10=1,15=1.5，20=2 
`dataStream` \- ：数据流：0：无，1：xon/xoff，2：硬件
`comType` \- ：串口通信方式 232、485、422不为这三个值默认为232
示例：
```
Com_m = M:1000:COM; //定义主机板号为1000的串口
// 设置串口Com_m第1路（即定义主机板号为1000的第一个串口）的
// 波特率为9600，数据位为8，无奇偶，停止位为1，数据流无，通信方式为232
SET_COM(Com_m,1,9600,8,0,10,0,232);
```

#  SEND_COM
```text
void SEND_COM(String dev,int channel, String str)
```
功能：串口数据发送
**Parameters:**
`dev` \- ：串口设备
`channel` \- :设备通道号
`str` \- ：串口数据,支持三种格式。
1：直接透传字符串数据（将字符串原封不动发往串口），
2：转换16进制字符串（遇到0x或0X打头开始的字符串，将字符串作为16进制的字符串进行转换后发送。如发送0x3132，串口收到为”12”的字符串）
3：字符串数据和16进制数据混合发送.
示例：
```
Com_m = M:1000:COM; //定义主机板号为1000的串口
SEND_COM(Com_m,1,”1234”); // 向主机板的1路串口发送字符串“1234”
SEND_COM(Com_m,1,”0x31323334”); // 向主机板的1路串口发送字符串“1234”
 
```
需要混合发送时请使用 **COMPOSE_COM** 函数。

# SOMPOSE_COM
```text
  String  SOMPOSE_COM(String[] comData)
```
功能：将10进制字符串和16进制字符串组合
**Parameters:**
comData：10进制或者16进制字符串
**Return Value** ：
组合起来的字符串。
示例：
```
DEFINE_VARIABLE
string str;
string ComData[2]; //定义一个string数组，数组大小根据需要定义
DEFINE_START
ComData[0] = "abcd"; //10进制字符串 ComData[1] = "0x313233"; //16进制字符串
str = COMPOSE_COM( ComData ); //
Com_m = M:1000:COM; //定义主机板号为1000的串口
SEND_COM(Com_m,1, str ); // 向主机板的1路串口发送字符串 "abcd123"。
```
