# GET_MATRIX_M
```text
  void GET_MATRIX_M(int chanel)
```

功能：获取矩阵输入通道状态
**Parameters:**
`Chanel` \- :通道号
示例：
```
int matrix_out_status;
matrix_out_status = GET_MATRIX_M(1);//查询矩阵
```
* * *
# SET_MATRIX_M
```text
void SET_MATRIX_M(int out, int in)
```
功能：切换矩阵
**Parameters:**
`out` \- :输出通道
`in` \- :输入通道
示例：
SET_MATRIX_M(1,3);//设置矩阵
* * *
**示例：******
```
DEFINE_DEVICE
	tp = T:10:TP;
	tp1 = T:11:TP;
DEFINE_COMBINE
	[ tp, tp1 ];
DEFINE_CONSTANT
DEFINE_VARIABLE
	int matrix_out_status;
DEFINE_FUNCTION
DEFINE_TIMER
DEFINE_START
DEFINE_EVENT
	BUTTON_EVENT(tp,1);
	{
		PUSH()
		{
			matrix_out_status = GET_MATRIX_M(1);//查询矩阵
		}
	}
	BUTTON_EVENT(tp,17);
	{
		PUSH()
		{
			SET_MATRIX_M(1,3);//设置矩阵
		}
	}

DEFINE_PROGRAME
```
