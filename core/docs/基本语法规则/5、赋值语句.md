**赋值语句**

赋值是改变一个变量的值和改变表域的最基本的方法。其格式为：变量 = 表达式；
用例如下：
```
DEFINE_VARIABLE  
 // 定义变量  
 int a;  
 double  b;  
 boolean c;  
 string d;  
 char e;  
 byte f;  
   
DEFINE_START  
 // 给变量赋值  
 a = 1;  
 b = 2.8 + 1;  
 c = true;  
 d = "hello world";  
 e = 'h';  
 f = 0;  
```
也可以在定义变量的时候赋值，用例如下： 
```
DEFINE_VARIABLE  
 // 定义变量并直接给变量赋初值  
string      a = "Hello World";  
int          b = 100;  
double   c = b + 5;  
boolean d = (true && false);  
char       e = 'g';  
byte       f = 1;
```

**注意：赋值时只能给某一类型的变量赋值为该类型的值，不能是其他类型的值。**
