函数是具有某特点功能并可以重复使用的一段代码。如果在程序中重复出现某段相同功能的代码，那么就应该把这些代码写成一个函数，并在原来出现这些代码的地方调用它。
## 1、函数的定义
```
返回值类型 函数名(参数1,参数2,...,参数n ){  
    函数体  
}
```
**(1) 返回值类型**
一个函数可以有返回值，返回值的类型在函数定义时有 “ 返回值类型” 定义。“ 返回值类型 ”只能是以下这几个数据类型之一：“void”、“int”、“char”、“double”、“byte”、“boolean”、“string”。
其中“void”是函数返回值特有的类型，表示空返回值，及表示函数没有实际的返回值。
**(2) 函数名**
函数名的命名规则与变量的命名规则一样， 具体请参考 ” [类型和值](1、类型和值.htm) “ 里标识符的命名规则 。
**(3) 参数**
一个函数可以有若干个参数，也可以没有参数。一个参数的完整定义是 “类型 参数名”，如果有多个参数，则每个参数之间用逗号“,”隔开。
参数与变量类型一样，只能使用“int”、“char”、“double”、“byte”、“boolean”、“string”中的某一个。
参数名命名规则与变量的命名规则一样， 具体请参考 ” [类型和值](1、类型和值.htm) “ 里标识符的命名规则 。
**(4)函数值是返回**
对于返回值不为void的函数来说，必须有一个 “return” 语句来返回相应类型的值。“return”语句可以放在函数体的任何地方。返回值为void的函数也可以使用 "return" 语句。
具体请参考下面的例子。
## 2、函数定义例子
**(1) 返回值为空、没有参数的函数**
函数fun1输出字符串“fun1”
```
void fun1(){   
    TRACE("fun1");  
}
```
假设在DEFINE_VARIABLE里定义的整形变量n
函数fun2通过加1或减1的方法把变量n的值向靠近10的方向调整。
```
void fun2()  
{  
    if( n > 10 )  
    {  
        n = n -1;  
        return;  
    }  
    else if( n < 10 )  
    {  
        n = n + 1;  
        return;  
    }  
}
```

**(2) 有一个参数的函数**
函数Square对通过参数传过来的整数n计算其平方数并返回计算结果。
```
int Square( int n ){    
    return n*n;   
}
```
函数GetWage把通过参数传过来的浮点数base作为工资基数，乘以倍率0.92得出最终应发工资并返回计算结果。
```
double GetWage( double base ){    
    double money = 0;  
    double rate = 0.92;  
    money = base * rate;  
    return money;  
}

```
函数PrintComputerName把通过参数传过来的字符串computer_name 输出。
```
void PrintComputerName( string computer_name ){   
    TRACE( "Computer Name: " + computer_name );  
}
```

**(3) 有多个参数的函数**
函数Add计算并返回参数a1+a2的和。
```
int Add( int a1, int a2 ){   
    return a1 + a2;  
}
```
函数Max查找参数a1、a2、a3中的最大值，并返回该最大值。
```
int Max( int a1, int a2, int a3 ){   
    int max = 0;  
   
    if( a1 > a2 )  
        max = a1;  
    else  
        max = a2;  
   
    if( a3 > max )  
        max = a3;  
   
    return max;  
}
```

## 3、函数调用
对以上例子定义的函数，其调用示例如下：
**(1) 没有参数的函数的调用**
fun1的调用。
```
fun1();
```
fun2的调用。
```
fun2();

```
**(2) 有参数的函数的调用**
函数Square的调用
```
int s = 0;   
s = Square(20);         // 计算20的平方保存在变量s中   
   
int a = 4;   
s = Square(a);           // 计算a的平方保存在变量s中    
s = Square(a+3);       // 计算a+3的平方保存在变量s中  
```

函数GetWage的调用
```
double money = GetWage( 1000 );       // 计算基数为1000的工资，结果保存在变量money中
```

 函数PrintComputerName的调用
```
PrintComputerName("IBM ThinkPad");
```
函数Add的调用
```
int sum = 0;   
sum = Add(10,20);         // 计算10+20的和保存在变量sum中  
Add(sum,30);                 // 计算10+20+30的和，不保存计算结果    
```
函数Max的调用
```
int num1 = 10;   
int num2 = 4;   
int num3 = 109;  
int max_num = Max(num1,num2,num3);         // 取num1,num2,num3中的最大值并保存在变量max_num中    
   
max_num  = Max(Max(1,2,3),4,5);                  // 取1,2,3,4,5中的最大值并保存在变量max_num中，**注意如何使用一个函数的返回值作为另一个函数的参数。**  

```
**(3) 返回值的使用**
对于返回值不是void的函数，都可以使用一个与返回值类型相同的变量来保存函数的返回值，如上面“函数调用”的例子所示。
