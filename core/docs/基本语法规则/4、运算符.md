**运算符**
**1.1、算术运算符**
       算术运算符是针对数值类型操作数进行的运算。系统提供的算术运算符列于表1-1
表1-1    算术运算符

| **符号** | **运算** | 例     | **功能**     |
| ------ | ------ | ----- | ---------- |
| +      | 加号     | a + b | 求a与b相加的和   |
| —      | 减号     | a - b | 求a与b相减的差   |
| *      | 乘号     | a * b | 求a与b相减的积   |
| /      | 除号     | a / b | 求a除以b的商    |
| %      | 模数     | a % b | 求a除以b所得的余数 |

**注意：(1)只有整数类型(int)的数据才能够进行取余运算。**
         **(2)两个整数类型的数据做除法时，结果是截取商数的整数部分，而小数部分被截断。**

**1.2、关系运算符**
关系运算是比较两个数据之间的大小关系的运算，系统提供的关系运算符列于表1-2
表1-2  关系运算符

| **符号** | **运算** | **例**  | **功能**      |
| ------ | ------ | ------ | ----------- |
| ==     | 等于     | a == b | 判断a是否等于b    |
| !=     | 不等于    | a != b | 判断a是否不等于b   |
| >      | 大于     | a > b  | 判断a是否大于b    |
| <      | 小于     | a < b  | 判断a是否小于b    |
| >=     | 大于等于   | a >= b | 判断a是否大于或等于b |
| <=     | 小于等于   | a <= b | 判断a是否小于或等于b |

**1.3、逻辑运算符**
逻辑运算是针对布尔型数据进行的运算，运算的结果仍然是布尔型量。系统提供的逻辑运算列于表1-3。
表  1-3 逻辑运算符

| **符号** | **运算** | **例**  | 运算规则            |
| ------ | ------ | ------ | --------------- |
| &      | 位逻辑与   | a & b  | a，b都真时结果才为真     |
| \|     | 位逻辑或   | a \| b | a，b有一个假结果都为假    |
| !      | 逻辑非    | !x     | x真时结果为假，x假时结果为真 |
| &&     | 逻辑与    | a && b | a，b都真时结果才为真     |
| \|     | 逻辑或    | a \| b | a，b都假时结果才为假     |
| ^^     | 逻辑异或   | a ^^ b | a,b同真假时加过为假     |

**1.4、位运算符**
位运算时对操作数以二进制比特位为单位进行的操作和运算，位运算的操作数和结果都是整型量。系统提供的位运算符列于表1-4。
表1-4  位运算符

| **符号** | **运算** | **例**  | **功能**    |
| ------ | ------ | ------ | --------- |
| ~      | 位反     | ~x     | 将x按比特位取反  |
| >>     | 右移     | x >> a | x各比特位右移a位 |
| <<     | 左移     | x << a | x各比特位左移a位 |

**1.5、特别符号**

| **符号** | 运算    | **功能**           |
| ------ | ----- | ---------------- |
| { }    | 括号运算符 | 多条命令集的结合，多用在函数方面 |
| [ ]    | 数组运算符 | 用于数组             |
| //     | 注释运算符 | 注释一行             |
| /**/   | 注释运算符 | 注释一段             |
| ;      | 分号    | 一个语句的结束          |

**2、运算符的优先级和结合性**

运算符的优先级决定了表达式中不同运算执行的先后顺序。结合性决定了并列的相同运算的先后执行顺序。系统运算符的优先级见表2-1

| **优先级** | **描述** | 运算符       | 结合性 |
| ------- | ------ | --------- | --- |
| 1       | 最高优先级  | [ ] { }   | 左/右 |
| 2       | 单目运算   | ~ !       | 右   |
| 3       | 算术乘除运算 | * ? %     | 左   |
| 4       | 算术加减运算 | + -       | 左   |
| 5       | 移位运算   | >> <<     | 左   |
| 6       | 大小关系运算 | > < >= <= | 左   |
| 7       | 相等关系运算 | == !=     | 左   |
| 8       | 按位与    | &         | 左   |
| 9       | 逻辑异或运算 | ^^        | 左   |
| 10      | 按位或    | \|        | 左   |
| 11      | 逻辑与    | &&        | 左   |
| 12      | 逻辑或    | \|        | 右   |
| 13      | 赋值     | =         | 右   |
