## 数组
数组是一类数据（如：整数型数据int，或字符型数据char等）的集合。
# 1、数组的定义
与变量定义类似，定义一个数组除了要指明数据类型和数组名外还需要指明数组的大小，即数组可以存放数据的个数。
数组的定义格式如下：
数据类型 数组名[数组大小]；
**数据类型** 只能是以下这几个数据类型之一：“int”、“char”、“double”、“byte”、“boolean”、“string”。
**数组名** 与变量的命名规则一样，具体请参考 ”[类型和值](1、类型和值.htm)“ 里标识符的命名规则 。
**数组大小** 只能是整形常数，如：10、100等。
**注：这里数组是动态数组，该大小只是定义分配的大小，在处理过程中可以改变。可以理解为指针变量，不同的处理指向不同的地址**
如以下定义的是一个大小为10个整形数的数组：
```
DEFINE_VARIABLE  
// 定义有10个元素的整形数组a  
int a[10];
```
# 2、数组的访问
数组的访问格式如下：
```
数组名[下标]
```
下标 可以是整形常数或整形变量，它指明了要访问数组中的第几个数据。下标的取值范围为：0~数组大小-1，即第1个元素的下标是0，第2个元素的下标是1...余此类推。如上面定义的数组“int a[10];”其下标取值范围是0~9。
对上面定义的数字进行访问：
```
DEFINE_START  
a[0] = 9;                  // 把数组的第1个元素赋值为 9  
a[i] = 5;                   // 把数组的第i+1个元素赋值为 5 （假设已定义了整形变量“i”，注意：应该保证“i”的值在下标合法范围0~9内）  
a[5+i] = a[0] + a[i];   // 把数组的第1个与第i+1个元素的和赋给第i+6个元素
```
