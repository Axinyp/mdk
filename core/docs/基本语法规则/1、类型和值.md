## 1、基本数据类型

| 类型名称   | 关键字     | 占用空间(字节) | 取值范围                             | 默认值   |
| ------ | ------- | -------- | -------------------------------- | ----- |
| 字节型    | byte    | 1        | -128—127                         | 0     |
| 整型     | int     | 4        | -2147483648—2147483647           | 0     |
| 双精度浮点型 | double  | 8        | -1.79769313E+308—1.79769313E+308 | 0.0   |
| 字符型    | char    | 1        | 0—65535                          | 0     |
| 布尔型    | boolean |          | true或false                       | false |
| 字符串    | string  |          |                                  | ""    |
| 空类型    | void    |          |                                  |       |
## 2、标识符
任何一个变量、常量、函数、数组和设备都有一个名字，这个名字就是标识符。标识符可以由编程者自由指定，但是需要遵循一定的语法规定。系统对于标识符的定义有如下的规定：
* (1)字母或下划线开头,由字母,数字,下划线组成的字符串
* (2)保留字 不能当作标识符。保留字包括关键字，系统对象名等。
* (3)标识符是大小写敏感的。
* (4)WAIT语句的名字也必须符合该要求,即WAIT语句的名字也是一个标识符。
**下是合法的标识符示例：**
```
dog                     // 以字母开头，由字母组成  
_dog                   // 以下划线开头，由字母组成  
car2                    // 以字母开头，由字母和数字组成  
_2_wheel_          // 以下划线开头，由字母、下划线和数字组成
```
**以下是不合法的标识符示例：**
```
2dog                   // 不能以数字开头  
do*g                   // 出现了字母、数字、下划线以外的字符--星号“*”，不合法  
do g                    // 出现了字母、数字、下划线以外的字符--空格“ ”，不合法  
while                   // 不能以系统关键字作为标识符  
BUTTON           // 不能以系统对象名作为标识符
```
**注意：由于标识符对大小写是敏感的，“dog”与“Dog”将被认为是不同的标识符。**
