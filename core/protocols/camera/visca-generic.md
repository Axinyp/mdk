# 摄像机 - VISCA 协议 (索尼/松下/华为等)

## 基本信息
- 设备类型：摄像机
- 通信方式：串口RS232（菊链多台时用 RS232 级联）
- 波特率：9600（部分型号 38400）
- 数据位/停止位/校验：8/1/无

## 设备声明
```
SET_COM(M_COM, ch, 9600, 8, 0, 10, 0, 232);
```

## 指令表（以摄像机地址1为例，地址N则首字节改为 0x8N）
| 功能 | 指令（HEX） | 说明 |
|------|------------|------|
| 开机（地址1） | `0x8101040000FF` | CAM_Power On |
| 关机（地址1） | `0x8101040100FF` | CAM_Power Off |
| 开机（地址2） | `0x8201040000FF` | |
| 关机（地址2） | `0x8201040100FF` | |
| 变焦放大 | `0x81010407{速度}FF` | 速度 0x02-0x07 |
| 变焦缩小 | `0x81010406{速度}FF` | |
| 变焦停止 | `0x81010400FF` | |
| 云台上 | `0x810106010E0E0301FF` | Pan=0x0E Tilt=0x0E |
| 云台下 | `0x810106010E0E0302FF` | |
| 云台左 | `0x810106010E0E0103FF` | |
| 云台右 | `0x810106010E0E0203FF` | |
| 云台停止 | `0x81010601000003FF` | |
| 预置位存储 | `0x8101043F01{位号}FF` | 位号 0x00-0x0F |
| 预置位调用 | `0x8101043F02{位号}FF` | |

## 代码示例
```
// 开机（摄像机1）
SEND_COM(M_COM, ch, "0x8101040000FF");

// 变焦放大（速度2）
SEND_COM(M_COM, ch, "0x81010407{02}FF");

// 调用预置位1
SEND_COM(M_COM, ch, "0x8101043F0201FF");
```
