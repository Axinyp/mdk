# 窗帘电机 - 通用 RS485 协议

## 基本信息
- 设备类型：窗帘电机
- 通信方式：串口RS485
- 波特率：9600
- 数据位/停止位/校验：8/1/无

## 设备声明
```
// 通过 TR-0740S 扩展模块的串口连接窗帘电机
SET_COM(TR_0740S_COM2, 2, 9600, 8, 0, 10, 0, 485);  // 窗帘1
SET_COM(TR_0740S_COM3, 2, 9600, 8, 0, 10, 0, 485);  // 窗帘2
```

## 指令表（0xFFAA 协议帧格式）
| 功能 | 指令（HEX） | 帧说明 |
|------|------------|--------|
| 开（展开） | `0xFFAA0400110213` | FF AA 04 00 11 02 13 |
| 停 | `0xFFAA0400110314` | FF AA 04 00 11 03 14 |
| 关（收起） | `0xFFAA0400110112` | FF AA 04 00 11 01 12 |

## 帧格式说明
```
FF AA   — 起始符
04      — 数据长度 (4字节)
00      — 设备地址 (广播)
11      — 功能码 (电机控制)
01/02/03 — 动作 (01=关, 02=开, 03=停)
校验位  — 前所有字节之和的低字节
```

## 代码示例
```
// 窗帘开（展开）
SEND_COM(TR_0740S_COM2, 2, "0xFFAA0400110213");

// 窗帘停
SEND_COM(TR_0740S_COM2, 2, "0xFFAA0400110314");

// 窗帘关（收起）
SEND_COM(TR_0740S_COM2, 2, "0xFFAA0400110112");

// 注意：如果电机安装方向相反，开/关指令需互换
// 例如 401 案例中 COM2 和 COM3 的开关方向相反：
SEND_COM(TR_0740S_COM2, 2, "0xFFAA0400110213");  // COM2 开
SEND_COM(TR_0740S_COM3, 2, "0xFFAA0400110112");  // COM3 开（方向相反）
```

## 适用型号
- 通用 RS485 窗帘电机（0xFFAA 协议帧格式）
- MKControl 生态中常见窗帘电机
